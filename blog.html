Important FAQ

What is the Document Object Modal (DOM) ?

The DOM represents a Document with a logical tree .Each branch of the tree ends in a node , and each node contains objects.

How do you select an element from the DOM ?

The easiest way to access a single element in the DOM is by its unique ID.

What is event delegation in the context of the DOM , Why is it useful ?

Event delegation is a design pattern in JavaScript used to efficiently manage and handle events on multiple child elements by attaching a single event listener to a common ancestor element.
